# SQL Injection Demonstration Project

This project demonstrates SQL Injection vulnerability and its prevention.

## Contents
- Vulnerable login system (SQL Injection possible)
- Secure login system (Prepared Statements)
- Admin and User dashboards
- Educational ethical hacking demonstration

## Requirements
- XAMPP (Apache + MySQL)
- PHP 7+
- Web browser

## How to Run
1. Clone the repository
2. Move the project to `htdocs`
3. Import the database (`sqli`)
4. Access via browser:
   - Vulnerable: http://localhost/vulnerable_site/login.php
   - Secure: http://localhost/secure_site/login.php

## Disclaimer
This project is for educational purposes only.
